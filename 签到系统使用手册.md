## 系统简介

值班签到系统用于记录公司各部门值班人员的签到情况，支持：
- **在线签到**：通过手机或电脑进行签到
- **权限管理**：区分普通用户和管理员
- **实时同步**：签到数据自动同步到金山云文档
- **数据导出**：支持历史数据查询和导出

### 支持的登录方式
1. **OA账号登录**：公司员工使用OA账号密码
2. **外委账号登录**：外包人员使用专用账号
3. **管理员账号**：本地管理员账号

## 访问签到系统

### 方法一：局域网访问（推荐）

1. **确保连接公司内网**
   - 连接公司WiFi或有线网络
   - 确保能访问内网资源

2. **打开浏览器**
   - 推荐使用 Chrome、Edge、Firefox 等现代浏览器
   - 不建议使用IE浏览器

3. **输入访问地址**
   ```
   http://[服务器IP]:3000
   ```
   
   例如：`http://192.168.1.100:3000`
   
   > **注意**：具体IP地址请询问系统管理员

### 方法二：本机访问（服务器上）

如果在签到服务器上操作：
```
http://localhost:3000
```

### 方法三：手机访问

1. 手机连接公司WiFi
2. 打开手机浏览器（Chrome、Safari等）
3. 输入相同的访问地址

## 登录系统

### 首次访问

打开签到页面后，系统会自动跳转到登录页面：

![登录页面](登录页面示意图)

### 登录方式选择

#### 方式一：OA账号登录（公司员工）

1. 选择登录方式：**"OA账号登录"**
2. 输入信息：
   - **账号**：你的OA账号（工号）
   - **密码**：你的OA密码
3. 点击 **"登录"** 按钮
4. 登录成功后自动跳转到签到页面

> **提示**：OA账号密码与公司其他系统（如邮箱、OA系统）一致

### 退出登录

页面右上角显示当前用户名和 **"退出登录"** 按钮：

1. 点击页面右上角的 **"退出登录"** 按钮
2. 确认退出后，系统会清除登录状态并返回登录页面

> **自动退出**：如果30分钟内没有任何操作，系统会自动退出登录，需要重新登录才能继续使用。

#### 方式二：外委账号登录（外包人员）

1. 选择登录方式：**"外委账号登录"**
2. 输入信息：
   - **账号**：外委账号（由管理员分配）
   - **密码**：外委账号密码
   - **真实姓名**：你的真实姓名（必填）
3. 点击 **"登录"** 按钮
4. 登录成功后自动跳转到签到页面

> **注意**：
> - 外委账号需要提前向管理员申请
> - 首次登录必须填写真实姓名
> - 部门信息会自动填充，无需修改

#### 方式三：管理员登录

管理员使用专用账号登录，具有补签、修改等高级权限。

## 进行签到

### 签到页面说明

登录成功后进入签到页面，页面分为几个区域：

```
┌─────────────────────────────────────────┐

│                                         │
│  【公司值班】                            │  ← 部门分类
│  ┌─────────┐ ┌─────────┐               │
│  │ 公司领导 │ │ 公司中层 │               │  ← 签到按钮
│  └─────────┘ └─────────┘               │
│  ┌─────────┐                            │
│  │ 公司干部 │                            │
│  └─────────┘                            │
│                                         │

│  │  库管   │ │         │               │

│                                         │
│  ... 其他部门 ...                        │
│                                         │
└─────────────────────────────────────────┘
```

### 签到步骤

#### 普通用户签到

1. **找到自己的部门**
   - 根据部门分类（公司值班、计划部、运行部等）
   - 找到自己的岗位名称

2. **点击签到按钮**
   - 点击对应的签到按钮（如"公司领导"、"库管"等）
   - 系统会自动记录：
     - 签到时间
     - 你的姓名
     - 部门信息
     - IP地址

3. **确认签到成功**
   - 按钮颜色变化（通常变为绿色）
   - 显示签到时间和姓名
   - 页面顶部显示成功提示

> **提示**：
> - 每个岗位只能有一人签到
> - 如果该岗位已有人签到，会显示他人姓名
> - 如需更换人员，请联系管理员

#### 外委人员签到

外委人员签到流程与普通用户相同，但注意：

1. **部门自动填充**
   - 登录时选择的部门会自动填充
   - 无需手动选择部门

2. **填写真实姓名**
   - 首次登录时已填写姓名
   - 签到时会自动使用该姓名

3. **只能签到自己部门**
   - 外委账号只能签到指定部门
   - 不能签到其他部门

## 管理员操作

### 管理员权限

管理员账号具有以下特殊权限：

1. **补签功能**
   - 为忘记签到的员工补签
   - 可以修改签到记录

2. **代签功能**
   - 替他人进行签到
   - 适用于特殊情况

3. **删除记录**
   - 删除错误的签到记录
   - 清空某岗位的签到

4. **查看所有记录**
   - 查看历史签到数据
   - 导出签到报表

### 如何进行补签

1. **登录管理员账号**

2. **找到需要补签的部门**

3. **长按或右键点击签到按钮**
   - 普通点击是自己签到
   - 长按/右键会弹出补签菜单

4. **选择"补签/修改"**

5. **填写补签信息**
   - 姓名：签到人员姓名
   - 工号：签到人员工号（可选）
   - 电话：联系电话（可选）
   - 签到时间：实际签到时间

6. **点击确认**

7. **系统记录为补签**
   - 记录会标记为"管理员补签"
   - 不影响正常签到统计

## 查看签到记录

### 方式一：签到页面实时查看

在签到页面可以直接看到：
- **已签到岗位**：显示绿色，有姓名和时间
- **未签到岗位**：显示灰色或白色，显示"-"

### 方式二：金山云文档查看（推荐）

签到数据会自动同步到金山云文档，可以通过以下方式查看：

#### 访问云文档

1. **打开浏览器**
2. **访问金山文档**
   - 网址：`https://www.kdocs.cn` 或 `https://365.kdocs.cn`
3. **登录金山账号**
   - 使用公司分配的金山文档账号
4. **找到签到表格**
   - 在我的文档或共享文档中找到
   - 表格名称通常为"值班签到记录"或类似名称

#### 云文档表格结构

云文档包含两个工作表：

**表1：历史签到数据**
- 包含所有日期的签到记录
- 可以按日期筛选
- 支持导出为Excel

**表2：当天签到数据**
- 仅显示当天的签到情况
- 实时更新（签到后约5秒内同步）
- 方便查看当日值班情况

#### 表格字段说明

| 字段 | 说明 | 示例 |
|-----|------|------|
| 日期 | 签到日期 | 2026-05-18 |
| 部门 | 部门分类-岗位 | 公司值班-公司领导 |
| 姓名 | 签到人姓名 | 张三 |
| 工号 | 员工工号 | 12345 |
| 电话 | 联系电话 | 138****8888 |
| 签到时间 | 具体签到时间 | 2026-05-18 08:30:15 |
| 签到状态 | 已签到/未签到 | 已签到 |
| 是否迟到 | 20:00后算迟到 | 否 |

#### 数据筛选和查询

**按日期筛选：**
1. 点击表头的"日期"列
2. 选择"筛选"
3. 选择要查看的日期

**按部门筛选：**
1. 点击表头的"部门"列
2. 选择"筛选"
3. 输入部门关键词（如"公司值班"）

**导出数据：**
1. 点击右上角"..."菜单
2. 选择"导出为Excel"
3. 选择保存位置

### 方式三：询问管理员

如果需要特殊格式的报表或历史数据，可以联系系统管理员导出。

## 常见问题

### Q1: 无法访问签到页面

**可能原因和解决方法：**

1. **网络问题**
   - 检查是否连接公司内网
   - 尝试访问其他内网资源确认网络正常

2. **地址错误**
   - 确认输入的IP地址和端口正确
   - 询问管理员确认最新地址

3. **服务器未启动**
   - 联系管理员确认签到系统是否运行

### Q2: 忘记OA密码

- 联系公司IT部门重置OA密码
- 重置后即可使用新密码登录签到系统

### Q3: 签到时提示"该岗位已签到"

- 说明该岗位已有人签到
- 如需更换，请联系管理员删除原记录后重新签到
- 或联系管理员进行补签修改

### Q4: 外委账号无法登录

1. 确认账号密码正确
2. 确认已填写真实姓名
3. 联系管理员确认账号是否有效

### Q5: 签到后云文档没有更新

**正常情况：**
- 实时同步通常需要5-10秒
- 请刷新云文档页面查看

**如果长时间未更新：**
1. 检查网络连接
2. 确认云文档地址正确
3. 联系管理员检查同步服务状态

### Q6: 如何修改已签到记录

- 普通用户无法自行修改
- 需要联系管理员进行补签/修改操作

### Q7: 手机签到和电脑签到有区别吗

- 功能完全相同
- 手机界面会自动适配屏幕大小
- 建议使用Chrome或Safari浏览器

### Q8: 签到时间显示不对

- 签到时间以服务器时间为准
- 如果显示时间偏差较大，联系管理员校准服务器时间
