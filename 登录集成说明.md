# 签到系统登录集成说明

## 概述

为原有签到系统添加了企业 OA 登录保护功能。用户必须先登录才能访问签到页面。

## 文件说明

| 文件 | 说明 |
|------|------|
| `server.py` | 原签到系统（无登录保护） |
| `server_with_auth.py` | 带登录保护的签到系统 |
| `启动签到系统(带登录).bat` | 一键启动脚本 |
| `login/` | @login 认证服务目录 |

## 快速开始

### 方法一：使用批处理文件（推荐）

双击运行 `启动签到系统(带登录).bat`

### 方法二：手动启动

1. **启动认证服务**
   ```bash
   cd login
   python attendance_login_only.py
   ```

2. **启动签到系统**
   ```bash
   python server_with_auth.py
   ```

## 访问地址

- 签到系统：http://localhost:3000
- 认证服务：http://localhost:8001

## 使用流程

1. 打开 http://localhost:3000
2. 点击"企业账号登录"
3. 输入 OA 账号密码
4. 登录成功后自动进入签到页面

## 切换回无登录版本

如果需要使用原来的无登录版本：

```bash
python server.py
```

## 注意事项

1. 认证服务必须先启动
2. 默认使用本地认证服务（http://localhost:8001）
3. 如需修改认证服务地址，设置环境变量：
   ```bash
   set AUTH_SERVICE_URL=http://your-auth-server:8001
   ```
